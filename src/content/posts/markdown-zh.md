# 一级标题

段落之间用空行分隔

第二段 _斜体_，**粗体**，和 `等宽字体` 无序列表
看起来像这样：

- 这一项
- 那一项
- 另一项

注意——不考虑星号——实际的文本内容从第4列开始

> 引用块是
> 这样写的
>
> 如果您愿意，它们可以跨越多个段落

使用3个破折号表示破折号 使用2个破折号表示范围（例如，"it's all
in chapters 12--14"） 三个点...将被转换为省略号
支持 Unicode ☺

## 二级标题

这里有一个有序列表：

1. 第一项
2. 第二项
3. 第三项

再次注意实际文本从第4列开始（从左侧4个字符） 这里有一个代码示例：

    # 让我重申一下...
    for i in 1 .. 10 { do-something(i) }

正如您可能猜到的，缩进4个空格 顺便说一下，如果您愿意，可以使用分隔块代替缩进：

```
define foobar() {
    print "Welcome to flavor country!";
}
```

（这样更容易复制和粘贴） 您可以选择标记分隔块以让 Pandoc 进行语法高亮：

```python
import time
# 快速，数到十！
for i in range(10):
    # （但不要太快）
    time.sleep(0.5)
    print i
```

### 三级标题

现在是一个嵌套列表：

1. 首先，获取这些食材：

    - 胡萝卜
    - 芹菜
    - 扁豆

2. 煮一些水

3. 把所有东西都倒进锅里，并遵循这个算法：

        找到木勺
        揭开锅盖
        搅拌
        盖上锅盖
        将木勺危险地平衡在锅柄上
        等待10分钟
        转到第一步（或者完成后关闭炉灶）

    不要碰木勺，否则它会掉下来
